PROJETOS

Blog dos gatos

Blog desenvolvido com HTML e CSS durante o curso da Rocketseat. O projeto conta com layout responsivo, cores personalizadas, tipografia clara e interações visuais suaves, oferecendo uma experiência de navegação agradável e organizada..

Github Favorites

GitHub Favorites é uma aplicação interativa que integra dados em tempo real usando APIs. Desenvolvido com princípios de orientação a objetos, o código é limpo, modular e fácil de manter. Utilizando LocalStorage, a aplicação permite salvar informações localmente, oferecendo uma experiência personalizada ao usuário.

Rocketnotes

RocketNotes é uma aplicação React construída com Vite, Styled Components e Context API. Cada funcionalidade foi organizada em componentes separados e com rotas dinâmicas, garantindo código modular, fácil manutenção e navegação fluida. O projeto permite que usuários criem conta, façam login, adicionem novas notas e pesquisem suas anotações de forma rápida.

To-do list

To-Do List desenvolvido com React, estruturado em componentes reutilizáveis (Button, Header, Input e Task). O projeto permite adicionar, listar e gerenciar tarefas de forma simples e prática. A aplicação utiliza estilização modular com CSS Modules, garantindo organização e facilidade de manutenção, além de oferecer uma interface limpa e responsiva.
